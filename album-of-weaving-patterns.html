Translation and Encoding

Album of Weaving Patterns is a web tool that is a translation and encoding of a Russian weaving manual for multi-shaft looms that was published in 1912. It is a practice in archiving, as well as reinterpretion.

This project begain in 2013 with an email from my mom, after a PDF copy of the manual was sent to her.

I encoded these patterns as arrays of binary corresponding to the threading, tie-up, and treadling(s) of each pattern and compiled a dictionary of weaving specific terms. In creating this website, I learned how to read weaving drafts and rendered these arrays to generated previews of the fabrics.

I also created an editable interface, which allows new patterns to be created or existing ones to be modified.
